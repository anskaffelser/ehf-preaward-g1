== Profile ID

The profile ID identifies the process that the business document is part of. This EHF Tendering guide uses the identification system according to {CENBII}.

IMPORTANT: As this document is covering different profiles, please make sure that you use the correct profile according to the processes (and transactions) covered in the profile.

This document document is covering the following profiles, which is identified with the stated profile ID in the element `cbc:ProfileID`:

Profile BII47::
`urn:www.cenbii.eu:profile:bii47:ver3.0`

Profile BII47x::
`urn:www.cenbii.eu:profile:bii47x:ver3.0`
